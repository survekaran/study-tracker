# 📚 BrainTrack – Study Tracker Web App

A modern and intuitive study tracker web application designed to help students stay consistent, organized, and productive. BrainTrack allows users to plan, track, and analyze their study sessions efficiently.

---

## 🚀 Features

* 📅 **Daily Study Tracking**
  Log your study sessions and monitor consistency

* ⏱️ **Timer / Session Tracking**
  Track focused study time using a built-in timer

* 📊 **Progress Analytics**
  Visual insights into your study patterns and productivity

* 🎯 **Goal Setting**
  Set daily or weekly study targets

* 🧠 **Focus Mode**
  Distraction-free interface for deep work

* 📌 **Task Management**
  Add, update, and complete study tasks

* 🌙 **Modern UI**
  Clean, responsive, and user-friendly design

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/braintrack.git
cd braintrack
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
